# Ritz Debugger

JPDA Debugger library for clojure.

## Usage

The library provides various layers for using JPDA/JDI in clojure.

## Install

Add the following to your `:dependencies`:

```clj
[ritz/ritz-debugger "0.4.0-SNAPSHOT"]
```
