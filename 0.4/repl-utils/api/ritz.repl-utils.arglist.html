ritz.repl-utils.arglist documentation

Arglists for symbols and sexps

arglist

[kw-or-symbol namespace]

Given a keyword or symbol and a possibly nil namespace, return an arglist

arglist-at-terminal

[sexp terminal ns]

Returns an arglist and an index for the position of the expression containing the given terminal.

branch-for-terminal

[sexp terminal]

Returns the (reversed) branch path for a given unique terminal in an sexp.

indexed-sexps

[sexp terminal]

Returns an sequence of [expr index] with the index of the argument of expr containing the specified terminal.

paths

[tree]

Find all paths to leaves in the given tree
