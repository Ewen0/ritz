ritz.repl-utils.namespaces documentation

Namespace functions

aliased-dependencies

[ns]

Calculate the namespaces that are aliased for a given namespace.

dependencies

[ns]

Return the namespaces the given namespace has as dependencies.

dependency-comparator

[dependency-map]

A comparator for dependencies in a dependency map

dependent-on

[ns]

Return the namespaces that are dependent on the given namespace symbol.

direct-dependencies

[]

Calculate namespace dependencies based on namespace information. This reduces a map, from namespace symbol, to a set of symbols for the direct namespace dependencies of that namespace.

namespace-dependencies

[ns]

Calculate the namespaces that the given namespace depends on.

package-dependencies

[ns]

Returns the packages on which the namespace depends.

refered-dependencies

[ns]

Calculate the namespaces that are refered for a given namespace.

sorted-dependencies

[dependency-map]

Sort a dependency map so that all namespaces appear after the namespaces that they depend on.

transitive-dependencies

[direct-dependencies]

Given a direct dependencies, returns transitive dependencies

unuse

[ns from-ns]
[ns]

Remove all symbols from from-ns that have been refered from ns

with-var-clearing

[ns & body]

Marks all vars in a namespace, executes the body, then removes all vars that still have metadata.
