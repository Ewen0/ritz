apropos-doc

[ns name public-only? case-sensitive? prefer-ns]

Return a list of docs for matching symbols for name, restricted to ns if non-nil prefering symbols accessible from prefer-ns.

javadoc-url

[classname ns]

Added: 1.2

Searches for a URL for the given class name. Tries local-javadocs first, then remote-javadocs. Returns a string.
