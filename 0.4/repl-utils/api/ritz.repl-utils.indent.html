ritz.repl-utils.indent documentation

indentation-hash

[]

Provides a hash of the namespace data. This allows a heuristic so as to avoid scanning all symbols from all namespaces. Instead, we only check whether the set of namespaces in the cache match the set of currently defined namespaces.

indentation-update

[indent-context ns]

Update the indentation cache in connection and update emacs. If force is true, then start again without considering the old cache.
