ritz.repl-utils.class-browse documentation

Provides Java classpath and (compiled) Clojure namespace browsing. Scans the classpath for all class files, and provides functions for categorizing them. Classes are resolved on the start-up classpath only. Calls to 'add-classpath', etc are not considered.

Class information is built as a list of maps of the following keys: :name Java class or Clojure namespace name :loc Classpath entry (directory or jar) on which the class is located :file Path of the class file, relative to :loc

class-or-ns-name

[n]

Returns the Java class or Clojure namespace name for a class relative path.

classpath

[]

Return the classpath File's for the current clojure classloader.

classpath-urls

[]

Return the classpath URL's for the current clojure classloader.

file-for-url

[url]

Convert a URL to a File.

path-class-files

Returns a list of classes found on the specified path location (jar or directory), each comprised of a map with the following keys: :name Java class or Clojure namespace name :loc Classpath entry (directory or jar) on which the class is located :file Path of the class file, relative to :loc

scan-paths

[files]

Takes one or more classpath strings, scans each classpath entry location, and returns a list of all class file paths found, each relative to its parent directory or jar on the classpath.
