ritz.repl-utils.completion documentation

Completion functions

class-name-list

[ns]

Returns a list of class names for a given namespace

classes-on-path

[symbol-string]

Returns a list of Java class and Clojure package names found on the current classpath. To minimize noise, list is nil unless a '.' is present in the search string, and nested classes are only shown if a '$' is present.

largest-common-prefix

[a b]

Returns the largest common prefix of two strings.

method-name-list

[ns]

Returns a list of method names (with leading dot) for a given namespace

ns-name-list

[ns]

Returns a list of potential namespaces for a given namespace

public-var-name-list

[ns]

Returns a list of public var names for a given namespace

resolve-class

[sym]

Attempts to resolve a symbol into a java Class. Returns nil on failure.

static-member-name-list

[class]

Returns a list of static members for a given class

var-name-list

[ns]

Returns a list of all var names for a given namespace
