ritz.jpda.jdi-vm documentation

Virtual machine launch and event handling.

Provides a context for a VM with a :control-thread

launch-vm

[classpath cmd & {:as options}]

Launch a vm and provide a control thread. Returns a context map. The vm is in a suspended state when returned.

start-control-thread-body

[thread-name]

Form to start a thread for the debugger to work with. This should contain only clojure.core symbols. Strings can also cause quoting issues.
