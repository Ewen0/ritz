ritz.jpda.jdi documentation

JDI wrapper. The aim here is to work towards a clojure interface for JDI but is currently mainly a set of light wrapper functions.

Predicate to test if a frame is a clojure frame. Checks the for the extension of the frame location's source name, or for the presence of well know clojure field prefixes.

handle-event

Client code should implement this for event tyoes that it is interested in receiving. By default, events are ignored.

handle-event-set

[queue connected context f]

NB, this resumes the event-set, so you will need to suspend within the handlers if required.

invoke-method

[thread {:keys [threading disable-exception-requests], :or {threading invoke-single-threaded, disable-exception-requests false}, :as options} class-or-object method args]

Methods can only be invoked on threads suspended for exceptions. args is a sequence of remote object references.
