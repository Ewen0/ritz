ritz.jpda.jdi-clj documentation

Clojure execution over jdi. Uses a context map to to represent the vm.

RemoteObject

Protocol for obtaining a remote object reference

clojure-fn

[context thread options ns name n]

Resolve a clojure function in the remote vm. Returns an ObjectReference and a Method for n arguments.

clojure-fn-deref

[context thread options ns name n]
[context thread options ns name]

Resolve a clojure function in the remote vm. Returns an ObjectReference and a Method for n arguments.

control-eval

[context form]

Eval an expression on the control thread. This uses single-threaded invocation, since the control thread is suspended in isolation.

eval

[context thread options form]

eval a form on the remote machine and return a local value. Relies on the value being readable. If it is unreadable, then a string as output by pr-str on the remote machine is returned.

eval-to-string

[context thread options form]

Evaluate a form, which must result in a string value. The string is return as a local value.

eval-to-value

[context thread options form]

Evaluate the specified form on the given thread. Returns a Value.

exception-message

[context event]

Returns a local string containing the remote exception's message.

invoke-clojure-fn

[context thread options ns name & args]

Invoke a clojure function on the specified thread with the given remote arguments.

invoke-signature

[n]

Clojure invoke signature for the specified number of arguments

pr-str-arg

[context thread options arg]

Read the value of the given arg

read-arg

[context thread arg]

Read the value of the given arg

remote-call

[context thread options sym & args]

Call a function using thread with the given remote arguments.

remote-str

[context s]

Create a remote string

remote-thread

[context thread options form thread-options]

Start a remote thread. `thread-options are: - :name set the name of the thread - :daemon daemonise the thread

remote-thread-form

[form thread-options]

Returns a form to start a thread to execute the specified form.
