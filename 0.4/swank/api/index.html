ritz.swank.commands

Namespace for swank functions. Slime calls functions in the swank package, so we need to be able o lookup symbols with a swank/ prefix. This is solved by interning every slime function into this namespace, and providing a namespace alias for itself. The namespace alias is then used when looking up symbols via ns-resolve.

Public variables and functions:

  • defslimefn
  • slime-fn
