location

[[{:keys [file buffer source-form zip], :as buffer-path} {:keys [position offset line column function-name method source-path eof], :as buffer-position} hints]]

A source location. Only one key can be specifed for each map. From slime-goto-source-location docstring: ::= (:file ) | (:buffer ) | (:source-form ) | (:zip )

::= (:position ) ; 1 based (for files) | (:offset ) ; start+offset (for C-c C-c) | (:line []) | (:function-name ) | (:source-path ) | (:method . )
