Contrib for providing ritz specific functions

exception-filter-list

[filters]

Return a sequence of exception filters, ensuring that expressions are strings and not regexes.

line-breakpoint

[connection namespace filename line]

Set a breakpoint at the specified line. Updates the vm-context in the connection.

list-repl-source-forms

[connection]

List all the source forms entered in the REPL

resume-vm

[connection]

Resume the vm. If the vm becomes suspended for some reason, you can use this to unsuspend it

toggle-swank-logging

[connection]

Control logging level
