ritz.nrepl.pr-values documentation

pr-values

[ops]
[]

Middleware that returns a handler which transforms any :value slots in messages sent via the request's Transport to strings via pr, delegating all actual message handling to the provided handler.

Requires that results of eval operations are sent in messages in a :value slot.
