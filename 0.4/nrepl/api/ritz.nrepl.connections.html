ritz.nrepl.connections documentation

Track client connections

all-connections

[]

Return all connections.

connections

A map from session id to connection

pending-ids

A map from message id to messages from the client.

primary-connection

[]

Return the primary connection. This can be used to obtain a connection to use for event notifications in the absence of a client message.

promote-pending-connection

[id session-id]

Promote the pending connection for message id to a connection for the session-id. Returns the connection.
